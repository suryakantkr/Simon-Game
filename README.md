# Simon Game Clone 🎮

A clean, responsive implementation of the memory-building **Simon Game**, styled to match exact vector block parameters. The application tracks randomized operational patterns, tracks user inputs sequentially, increments complexity tiers, and registers high-fidelity keyboard triggers natively.

## 🕹️ Game Rules
1. **To Start:** Press any physical key on your keyboard.
2. **Observe:** The computer will flash a specific box panel. Remember that choice element ID.
3. **Replicate:** Click on the blocks matching the computer's output sequence string sequence exactly.
4. **Advance:** Correct patterns increment your progress tier (`Level 1 -> Level 2`). Missing a singular sequential step triggers a global **Game Over** visual environment shift.

---

## ✨ Features

* **Authentic Geometric Grid:** Accurately renders the target panel architecture with exact custom thick-border stroke variables (`div1` through `div4`).
* **Linear Array Comparison Processing:** Natively evaluates current index inputs against system arrays to safely verify progress or log failure instances instantly.
* **Micro-Interaction Animations:** Dynamic CSS classes handle block opacity triggers (`.flash`) and press depth mutations (`.pressed`) smoothly.
* **Seamless Reset System:** Soft-fails and clears session tracking metrics gracefully on active errors to let the user reset progress simply with a key hit.

---

## 🛠️ Technology Architecture

* **HTML5:** Modular block segmentation leveraging accessible container elements.
* **CSS3:** High-density borders, system serif variable rendering, dynamic operational animations, and micro-interaction states.
* **Vanilla JavaScript:** Structural event loops, event listeners, math random indexes, and conditional logic trees.
